# Doctor Appointment Booking

A full-stack web application for booking doctor appointments.

## Stack
- **Frontend**: React + Vite
- **Backend**: FastAPI (Python)
- **Database**: MySql

## Getting Started

### Backend
```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
